== Description ==

A Wordpress plugin that displays all the Landfills in Vaucluse.

Made with ReactJs, mapboxgl and Gutenberg.


== Installation ==

1. Clone the repo in wp-content/plugins

2. Put your API key in the wp-config.php file at the root of your WordPress folder like this : 
// API key for mapbox
define('REACT_APP_MAPBOX_ACCESS_TOKEN', 'yourApiKeyGoesHere');

Before this line:
/* That's all, stop editing! Happy publishing. */

3. Activate the plugin through the 'Plugins' screen in WordPress
