== Description ==

This is a wordpress plugin that displays an interactive map that shows all the landfills (private, public or assiociations) in the Vaucluse (France).

This plugin uses mapbox-gl and @wordpress/create-block librairies.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/carte-decheteries-vaucluse` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
